<h2 style="color: #d25349;"><i class="fa fa-folder-open-o"></i> Smart Ordering System - Node.js Version</h2>
<h6 style="margin-top: 5px; margin-bottom: 5px; text-align: left; color: #a69e8d;">현재 요식업계에서 주문벨, 종업원들의 주문기록 실수, 일시적인 주문 데이터 등의 문제점을 개선하고자 Family Restaurant인 아웃백을 예시로 개발한 통합 매장 관리 시스템</h6>
<h6 style="color: #674929;">
❖ 개발 기간
2014.09.01 ~ 2015.05.29

❖ 주요 역할 : 팀장
(1) 서버 구축 (NODE.JS, Express)
(2) 데이터베이스 개발(MySQL)
(3) 클라이언트 개발- 손님용 MOBILE WEB APPLICATION
- 관리자용 WEB CLIENT

Node.js로 서버를 구축한 뒤 고객 주문목록, 테이블목록, 주문메뉴목록, 
지점별 설명, 메뉴 등 레스토랑의 정보를 MySQL을 사용하여 저장

기존의 XML 보다 효율적인 데이터 전송을 위해 JSON을 이용하였으며
JavaScript와 jQuery, JSON과 Ajax를 기반으로 데이터 파싱을 구현

❖ 개발환경
Ubuntu 12.04 LTS, Express, Node.js, MySQL 5.5

❖ 사용 기술 • Tools
HTML, CSS, JavaScript, jQuery, Bootstrap
Linux vi 편집기, Node.js, MySQL, Xshell, Xftp, Github
